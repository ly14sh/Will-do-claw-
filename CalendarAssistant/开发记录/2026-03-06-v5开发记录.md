# 2026-03-06 开发记录 (v5版本)

## 需求背景

用户反馈：需要提升应用的状态栏视觉反馈体验，通过动态 SmallIcon 实现以下效果：
- 业务图标：区分快递 (📦) 和外卖 (🍔)
- 过程反馈：识别中显示扫描框 (📷)，分析完成显示星星 (✨)
- 网速胶囊：使用上下箭头 (⬆⬇) 替代 WiFi 图标

---

## 解决方案

### 模块一：状态栏图标资源

创建白色状态栏图标 (24dp, #FFFFFF)：

| 文件 | 用途 |
|------|------|
| `ic_stat_scan.xml` | 识别中（扫描框） |
| `ic_stat_sparkle.xml` | 分析完成（星星） |
| `ic_stat_success.xml` | 添加成功（勾选） |
| `ic_stat_package.xml` | 快递取件 |
| `ic_stat_food.xml` | 餐饮取餐 |
| `ic_stat_train.xml` | 火车出行 |
| `ic_stat_car.xml` | 打车出行 |
| `ic_stat_net.xml` | 网速胶囊（原 ic_stat_traffic 重命名） |

### 模块二：AI Prompt 优化

修改 `AiPrompts.kt` 的 `getUnifiedPrompt()` 和 `getPickupPrompt()` 方法：

- 快递格式：`【取件】号码|品牌|位置`
- 餐饮格式：`【取餐】号码|品牌|位置`

### 模块三：IconUtils 工具类

新建 `IconUtils.kt`，实现图标映射逻辑：

```kotlin
private fun isFoodPickup(description: String?): Boolean {
    return description?.startsWith("【取餐】") == true
}

fun getSmallIconForCapsule(capsule: CapsuleUiState.Active.CapsuleItem): Int {
    return when (capsule.type) {
        CapsuleService.TYPE_NETWORK_SPEED -> R.drawable.ic_stat_net
        CapsuleService.TYPE_PICKUP, CapsuleService.TYPE_PICKUP_EXPIRED -> {
            if (isFoodPickup(capsule.description)) R.drawable.ic_stat_food else R.drawable.ic_stat_package
        }
        // ...
    }
}
```

### 模块四：服务层集成

**CapsuleService：**
- 修改 `CapsuleUiState.kt`：添加 `description` 字段

## 修复问题

### 1. PickupUtils 正则表达式修复

**问题**：正则只匹配【取件】，不匹配【取餐】

**修复**：
```kotlin
// 修改前
private val PICKUP_PATTERN = Regex("【取件】([^|]+)\\|([^|]+)(?:\\|(.*))?")

// 修改后
private val PICKUP_PATTERN = Regex("【取(件|餐)】([^|]+)\\|([^|]+)(?:\\|(.*))?")
```

同时修复 groupValues 索引：
```kotlin
// 修改前
Triple(match.groupValues[1], match.groupValues[2], match.groupValues[3])

// 修改后
Triple(match.groupValues[2], match.groupValues[3], match.groupValues[4])
```

### 2. Kotlin 接口默认参数问题

**问题**：override 方法不允许指定默认参数

**修复**：移除接口默认参数，在实现类中指定默认值

### 3. Flyme 状态栏图标问题

**问题**：Flyme 胶囊的状态栏小图标未使用自定义图标

**修复**：使用传入的 `iconResId` 参数

## UI 优化

### 编辑日程 Dialog 改造

**需求**：在类型上显示所有 tag 类型，根据当前事件的 tag 给类型标注颜色，用户不可更改

**实现**：
- 显示 4 种 tag：日程、取件、火车、打车
- 使用 `primary` 主题色高亮当前选中的 tag
- 使用 `AssistChip` 组件，`onClick = {}` 空实现
- 使用 `border = null` 去除水波纹特效
- 使用 `horizontalArrangement = Arrangement.spacedBy(8.dp)` 等距分布

### 侧边栏水波纹特效关闭

**需求**：关闭侧边栏点击时的水波纹特效

**实现**：
```kotlin
val interactionSource = remember { MutableInteractionSource() }
.clickable(onClick = onClick, interactionSource = interactionSource, indication = null)
```
