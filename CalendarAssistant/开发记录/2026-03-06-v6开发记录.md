## 问题分析与修复

### 1. 网约车动态标题显示错误

**问题**：打车事件的动态标题显示的是车型（如"比亚迪海狮05EV"）而非车牌号（如"鄂ABT558"）

**原因分析**：
`TransportUtils.parseRide` 方法对 `parts` 数组的索引理解错误：
- 测试样例：`【用车】灰色|比亚迪海狮05EV|鄂ABT558`
- `parts[0]` = 灰色（颜色）
- `parts[1]` = 比亚迪海狮05EV（车型）
- `parts[2]` = 鄂ABT558（车牌）

原代码把 `parts[1]` 当作车牌显示，导致显示车型而非车牌。

**修复方案**：
```kotlin
// 修改前
parts.size >= 2 -> {
    val carModel = parts[0].ifBlank { "" }
    val licensePlate = parts[1].ifBlank { "" }
    mainDisplay = licensePlate
}

// 修改后
parts.size >= 3 -> {
    val color = parts[0].ifBlank { "" }
    val carModel = parts[1].ifBlank { "" }
    val licensePlate = parts[2].ifBlank { "" }
    mainDisplay = licensePlate
    subDisplay = "$color $carModel"
}
```

**修改文件**：`core/util/TransportUtils.kt`

### 2. 已完成事件被错误唤醒

**问题**：用户完成一个事件后刷新胶囊状态，之前已完成但未过期的另一个事件会被错误"唤醒"显示

**原因分析**：
`CapsuleStateManager` 过滤活跃事件时，只检查时间是否过期，没有排除 `isCompleted = true` 的已完成事件：
```kotlin
val isActive = now.isBefore(endDateTime) && !now.isBefore(effectiveStartTime)
```

**修复方案**：
```kotlin
val isActive = !event.isCompleted && now.isBefore(endDateTime) && !now.isBefore(effectiveStartTime)
```

**修改文件**：`core/capsule/CapsuleStateManager.kt`

### 4. 网速胶囊显示不全

**问题**：网速胶囊标题被截断，如 `↓11.0KB/..` 或 `↓133.6KB...`

**原因分析**：
1. Flyme 胶囊标题限制 10 字符
2. 原格式 `↓ 11.0 KB/s` 长度 11 字符，会被截断
3. `999 KB/s` 达 9 字符，接近限制

**修复方案**：

**4.1 去掉 ↓ 符号**
```kotlin
// CapsuleStateManager.kt
// 修改前
title = "↓ ${networkSpeed.formattedSpeed}"

// 修改后
title = networkSpeed.formattedSpeed
```

**4.2 优化格式 + 添加进位逻辑**
```kotlin
// NetworkSpeedMonitor.kt
private fun formatSpeed(bytesPerSecond: Long): String {
    return when {
        bytesPerSecond < 1024 -> "${bytesPerSecond}B/s"
        bytesPerSecond < 1024 * 1000 -> "${bytesPerSecond / 1024}KB/s"
        bytesPerSecond < 1024 * 1024 -> String.format("%.1fMB/s",
            bytesPerSecond / (1024.0 * 1024.0))
        else -> String.format("%.1fGB/s",
            bytesPerSecond / (1024.0 * 1024.0 * 1024.0))
    }
}
```

**显示效果对比**：

| 原网速 | 修改后 | 字符数 |
|--------|--------|--------|
| `↓ 512 B/s` | `512B/s` | 6 |
| `↓ 999 KB/s` | `999KB/s` | 7 |
| `↓ 1000 KB/s` | `1.0MB/s` | 6 |
| `↓ 1.5 MB/s` | `1.5MB/s` | 6 |
| `↓ 1.23 GB/s` | `1.2GB/s` | 6 |

**修改文件**：
- `core/capsule/CapsuleStateManager.kt`
- `service/capsule/NetworkSpeedMonitor.kt`
