# 2026-03-15 开发记录 (v20版本)

## 需求背景

用户反馈提示词导入/导出存在混淆：导入后 UI 仍显示“云端 v4”，自定义格式缺失字段时会被自动补齐；同时需要将文本/多模态提示词统一管理，支持全局提示头；并希望对 AI 失败原因进行细化提示（HTTP 状态/参数错误/网络等），避免“未识别到有效日程”的误导。

另有性能诉求：在 OpenAI 接口上关闭思考以提速，并提供可配置开关。

---

## 一、本次实现范围

### 1) 提示词结构升级与 v5 迁移

**修改文件**：
- `core/ai/AiPrompts.kt`
- `data/model/RemotePrompts.kt`
- `ui/page_display/settings/BackupSettingsPage.kt`
- `ui/viewmodel/MainViewModel.kt`

**核心改动**：
1. 新增 `promptHeader` 全局提示块（布局标记 + Title/description 规范），运行时自动拼接到所有提示词。
2. 导出格式调整：`promptHeader + schedulePrompt + pickupPrompt + mmUnifiedPrompt`（不再导出 userText）。
3. 导入解析支持 `promptHeader`，并兼容旧格式；`=== end ===` 解析修正。
4. 提示词版本提升至 v5，本地缓存版本 < 5 自动清理并写入新默认。
5. 移除 `unifiedPrompt` 旧字段与相关逻辑（统一走 `mmUnifiedPrompt` + `userTextPrompt`）。

### 2) 提示词来源与 UI 显示规则

**修改文件**：
- `core/ai/AiPrompts.kt`
- `ui/page_display/settings/BackupSettingsPage.kt`
- `ui/viewmodel/MainViewModel.kt`

**核心改动**：
1. 新增提示词来源标记：LOCAL / CLOUD。
2. 导入提示词强制标记为本地；云端更新标记为云端。
3. UI 显示逻辑调整：
   - 本地：只显示“本地”
   - 云端：显示“云端 vX”

### 3) OpenAI “关闭思考”开关（仅适配 OpenAI）

**修改文件**：
- `data/model/MySettings.kt`
- `ui/page_display/settings/PreferenceSettingsPage.kt`
- `ui/viewmodel/SettingsViewModel.kt`
- `data/model/ModelPayload.kt`
- `core/ai/ApiModelProvider.kt`
- `core/ai/RecognitionProcessor.kt`

**核心改动**：
1. 新增设置项 `disableThinking`（默认关闭），偏好页提供“快速模式（关闭思考）”开关，并标注仅适配 OpenAI。
2. OpenAI 端点检测（`openai.com` 或 `/v1/chat/completions`）命中时注入 `reasoning_effort: "low"`。
3. 若 400/422 且提示参数不兼容，自动重试一次并缓存“不支持”避免重复报错。

### 4) 失败原因细化与全链路提示

**修改文件**：
- `core/ai/ApiModelProvider.kt`
- `core/ai/RecognitionProcessor.kt`
- `ui/page_display/HomePage.kt`
- `service/floating/FloatingScheduleService.kt`
- `service/accessibility/TextAccessibilityService.kt`

**核心改动**：
1. API 调用返回结构化结果（成功/失败），携带 HTTP 状态与错误类型。
2. 统一错误映射规则：
   - 401/403 -> API Key 无效或无权限
   - 404 -> 服务地址(URL)无效或模型不存在
   - 400/422 -> 请求参数错误/模型不支持该参数
   - 超时/断网 -> 网络连接失败
3. 文本输入、OCR、以及多模态识别均显示“分析失败：原因 (code)”或“未识别到有效日程”。

### 5) 视觉请求序列化修复

**修改文件**：
- `core/ai/ApiModelProvider.kt`

**核心改动**：
1. 多模态请求 `JsonObject` 发送前转换为字符串，避免 `JsonLiteral` 序列化异常。

## 二、行为变化说明

1. 提示词导出文件新增 `promptHeader` 段；导入时会自动拼接全局提示到所有提示词。
2. 本地导入后 UI 显示“本地”，云端更新显示“云端 vX”。
3. 开启“快速模式（关闭思考）”仅对 OpenAI 请求生效；不兼容时自动降级重试。
4. 识别失败会显示具体原因（含 HTTP 状态码），不再统一显示“未识别到有效日程”。
